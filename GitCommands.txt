

Step 1 :	Check if git is already installed
			git --version
Step 2 :	download and install git
			git-scm.com
Step 3 :	Add your project to git
			git config --global user.email "yourgithub@email.com"
			git config --global user.name "yourgithubUserName"
Step 4 :	create an account on github
Step 5 :	copy the online URL to push the changes on remote (cloude)
Step 6 :	Commands
			git init
			git status
			git add <file-name or wildcards>
			git commit -m <"Message that what you have added/deleted/modified">
			git log
			git --help
		Add on Remote machine -	
			git remote add origin <https://github.com/pritipal1984/Repo1.git>
			git push -u origin master
			
Branch Management in GIT
Step 1 :	Create Branch    =====  git branch <"new branch name">
			git branch MyNewBranch
Step 2 :	Checkout Branch
			git checkout MyNewBranch
		Still online it shows one branch so add a new file and then check the git status 
			git status
Step 3 :	git add .
			git commit -m "Updated file GitCommands.txt"
			git push -u origin MyNewBranch
			git checkout master
Step 4 :	Merge new Branch in master branch
			git merge MyNewBranch
			git push -u origin master
			git checkout MyNewBranch
			git status
Step 5 :	To Delete Branch, you must checkout some other branch like master
			git checkout master
			git branch -d MyNewBranch				--To delete from local
			git push origin --delete MyNewBranch	--To delete from remote
